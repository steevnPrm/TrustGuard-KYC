# Architecture Decision Records (ADR)

Décisions d'architecture documentées et versionnées dans ce dossier.

## Format

Chaque ADR suit le modèle :

1. **Contexte** — problème ou besoin
2. **Décision** — choix retenu
3. **Conséquences** — impacts positifs et négatifs

Nommage : `NNNN-titre-court.md` (ex. `0001-record-architecture-decisions.md`).
