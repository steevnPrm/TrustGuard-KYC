# Getting Started

## Prérequis

- **Java 21+** (API Spring Boot)
- **Node.js 20+** et **npm** (frontend Next.js)
- **Docker** et **Docker Compose** (environnement local recommandé)
- **Git**

## Installation

```bash
git clone https://github.com/steevnPrm/TrustGuard-KYC.git
cd TrustGuard-KYC
```
